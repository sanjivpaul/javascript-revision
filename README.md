### Strings in javascript:
**String Methods**
#### 1. `Length` 
return the length of the string
``` js
const gameName = new String("sanjiv");
console.log(gameName.length); // 6
```
#### 2. `toUpperCase()` 
convert all strings to upper case
``` js
const gameName = new String("sanjiv");
console.log(gameName.toUpperCase()); // SANJIV
```
